# StudyClaner

Task and activity management application with a minimalist design.

## Features
- Task management (Add, list, update status)
- Real-time statistics
- Responsive dashboard
- Minimalist Black & White aesthetic

## Tech Stack
- Frontend: React + Vite
- Backend: Django (Remote)
- Database: Supabase

## Setup
1. Install dependencies: `npm install`
2. Configure `.env` with `VITE_API_URL`
3. Run development server: `npm run dev`
